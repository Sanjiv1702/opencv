# Fuzzy Image Processor - Streamlit App

## Features
- User login and registration (with Streamlit Authenticator)
- SQLite-based user database
- Simple fuzzy image edge detection using OpenCV
- Placeholder buttons for future social login (Google, GitHub, Twitter)

## Login Credentials
- **Username**: admin
- **Password**: admin

## How to Run Locally
```bash
pip install -r requirements.txt
streamlit run streamlit_app.py
```

## Recommended Deployment
Use [Streamlit Cloud](https://streamlit.io/cloud) and upload all files from this ZIP.
